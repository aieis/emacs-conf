# emacs-conf

## Startup

Run `M-x aieis/setup-init`

## Requirements

At least emacs-28

## TIPS
For older versions it may be necessary to add the gpg keys for GNU Elpa

```
    gpg --homedir ~/.emacs.d/elpa/gnupg --keyserver hkps://keyserver.ubuntu.com --receive-keys 645357D2883A0966
```

There is also the OpenPGP keyserver

`hkps://keys.openpgp.org`
